Autonomous Domains (Each Own Their Database), One Ontology and One Logical Database

The Access domain and the Aggregation domain each own and edit their own Turtle file. Neither reads the other's database. Yet because both use the same shared ontology and IRI naming convention, their data merges into a single queryable graph with nothing more than a union.
